Halaman PNM - Membuka di tab/frame yang sama

Halaman List - Membuka di tab baru

Halaman Paragraf - Membuka di frame parent

Halaman Heading - Membuka di seluruh jendela browser

Kesimpulan:

Dari percobaan di atas dapat disimpulkan bahwa:
- _self membuka link di halaman yang sama
- _blank membuka link di tab atau jendela baru
- _parent membuka link di frame induk
- _top membuka link di seluruh jendela browser, keluar dari semua frame
