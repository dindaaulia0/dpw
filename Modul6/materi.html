📚 Materi Belajar CSS Framework

📌 1. Apa itu CSS Framework?

CSS Framework adalah kumpulan kode CSS yang sudah jadi. Kita tinggal memakainya, tidak perlu membuat dari awal.

Contoh CSS Framework populer: Bootstrap, Tailwind CSS, Materialize, dan Bulma.

📌 2. Cara Memasang Bootstrap

Tambahkan kode berikut ke file HTML kamu:

<!-- Di dalam tag <head> -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Sebelum tag </body> -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

📌 3. Grid System Bootstrap

Bootstrap punya sistem 12 kolom. Contoh membuat 2 kolom:

<div class="container">
  <div class="row">
    <div class="col-md-6">Kiri</div>
    <div class="col-md-6">Kanan</div>
  </div>
</div>

📌 4. Komponen Bootstrap

Contoh tombol dan card:

<!-- Tombol -->
<button class="btn btn-primary">Klik Saya</button>
<button class="btn btn-outline-success">Sukses</button>

<!-- Card -->
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">Judul</h5>
    <p class="card-text">Isi card disini</p>
  </div>
</div>

📌 5. Membuat Website Responsif

Gunakan class col-md-6, col-sm-12 untuk mengatur tampilan di HP, tablet, dan komputer.

✅ Tips: Class "md" untuk layar sedang (tablet/komputer), "sm" untuk HP.
