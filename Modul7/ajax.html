Apa itu AJAX?

AJAX (Asynchronous JavaScript and XML) adalah teknik untuk mengirim dan menerima data dari server tanpa me-reload seluruh halaman web.

Cara kerja AJAX:

  1. User melakukan aksi (klik, isi form, dll).
  2. JavaScript mengirim request ke server secara asinkronus.
  3. Server memproses dan mengembalikan data.
  4. JavaScript memperbarui bagian halaman yang diperlukan.
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    document.getElementById('hasil').innerHTML = data.pesan;
  });

Apa itu jQuery?

jQuery adalah library JavaScript yang mempermudah manipulasi HTML, penanganan event, animasi, dan AJAX. Moto: "Write less, do more."

Cara pakai jQuery:

<script src="jquery.min.js"></script>

$("#elemen").hide();          // sembunyikan
$("#elemen").show();          // tampilkan
$("p").text("Teks baru");     // ubah teks
$("#elemen").css("color","red"); // ubah style
